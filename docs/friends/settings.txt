FRIENDS_USE_NOTIFICATION_APP - by default notification app is used if it is enabled in INSTALLED_APPS, if this setting is set to False notification app will not be used
default: True

SHOW_FRIENDS_OF_FRIEND - allow users to view list of friends of their friends
default: False

NOTIFY_ABOUT_NEW_FRIENDS_OF_FRIEND - if notification app is enabled and FRIENDS_USE_NOTIFICATION_APP is set to True and this setting is set to True users will be notified when one of their friend have new friend
default: False

NOTIFY_ABOUT_FRIENDS_REMOVAL - if notification app is enabled and FRIENDS_USE_NOTIFICATION_APP is set to True and this setting is set to True users will be notified when some of their friends removes them from friends
default: False
